# 労働衛生コンサルタント 口述試験対策システム【開発者版】

労働衛生コンサルタント（保健衛生）口述試験の対策用チャットボットです。
**この版には問題管理機能が含まれています。**

## 🎮 モード説明

| モード | 説明 |
|--------|------|
| **本編** | 基本→詳細→応用の3段階構成 |
| **追加問題編** | 補足的な知識を補強 |
| **復習フォルダ** | 5点以下の問題を復習 |
| **問題を管理・追加** | マークダウンを貼り付けて問題追加 |

## ➕ 問題の追加方法

### 方法1: ファイルから追加（推奨）
- **本編**: `data/questions.md` を編集
- **追加問題編**: `data/followup.md` を編集

### 方法2: アプリ内で追加
1. 「問題を管理・追加」をクリック
2. 「本編問題を追加」または「追加問題を追加」を選択
3. マークダウンをテキストエリアに貼り付け
4. 「追加」ボタンをクリック

## 📝 マークダウン形式

### 本編問題の形式

```markdown
## テーマ名

**Q1（基本問題）：**
問題文

**【模範解答】**
解答文

**【この問題で試験官が見ているポイント】**
- ポイント1
- ポイント2

---

**Q2（詳細問題）：**
...
```

### 追加問題の形式

```markdown
### Q：質問文

**A：** 回答文

---
```

## 🌐 GitHub Pagesでの公開方法

1. GitHubでリポジトリを作成
2. このフォルダの内容をプッシュ
3. Settings → Pages → Source: Deploy from a branch → main / root

## 📦 頒布用バージョンの作成

頒布用（管理機能なし）を作成する場合は、`index.html` から以下を削除：

```html
{/* 管理ボタン */}
<div className="mt-4 flex gap-2">
  <button onClick={() => setAppMode('admin')} ...>
    ⚙️ 問題を管理・追加
  </button>
</div>
```

## 📁 ファイル構成

```
/
├── index.html      # アプリ本体（管理機能付き）
├── data/
│   ├── questions.md   # 本編問題
│   └── followup.md    # 追加問題
└── README.md       # このファイル
```

## ⚠️ 注意事項

- インターネット接続が必要（AI採点にAPIを使用）
- 手動追加した問題はlocalStorageに保存
- ファイルから読み込んだ問題と手動追加は統合して出題
